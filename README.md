## Overview

ShopHub is designed as a single-page e-commerce application that runs entirely in the browser without requiring any backend or build tools.  
All user data, cart information, and orders are stored using browser LocalStorage, making it ideal for learning, demos, and portfolio projects.

---

## Features

- Home page with hero section and featured products
- Product listing with live data from FakeStore API
- Search, category filtering, and price filtering
- Product detail view
- Shopping cart with add, remove, and quantity update
- Checkout process with order summary
- User authentication (login and logout using LocalStorage)
- Admin panel for product management (add, edit, delete)
- Orders page with order history
- Basic rule-based AI chat assistant
- Persistent state using LocalStorage
- Fully responsive design

## Technology Stack

### Frontend
- HTML5

- JavaScript (ES6)
- React 18 (CDN-based)

### Tools & Services
- Font Awesome (icons)

> Note: The entire application is implemented in a single HTML file.

## Installation & Usage

1. Clone or download the repository
2. Open the file `shophub-production.html`
3. Launch it in any modern web browser (Google Chrome recommended)
4. The application will run locally without additional setup

## AI Chat Assistant

The AI Chat Assistant provides basic assistance related to:

- Offers and promotions
- Product navigation
- Order-related queries

The chatbot is rule-based and implemented entirely on the frontend.  
It can be extended to integrate real AI services such as OpenAI using a backend.
